## 🔖&nbsp; Sobre
	
O projeto **ZapRecall** é um site de flashcards que foi criado no bootcamp da **Driven** com o intuito de praticarmos todo conteúdo exibido durante a semana.

### Features

- ✔️ Selecinar baralho de cartas
- 🚧 Definir meta de cartas respondidas corretamente
- ✔️ Acessar pagina principal
- ✔️ Responder as cartas de um baralho
- ✔️ Receber resultado sobre as respostas

## 🗂 Como baixar o projeto

```bash
	
    # Instalar o npm
    #  No Windows acesse
    $  https://nodejs.org/en/
	
    #  No linux ou wsl
    $  wget -qO- https://raw.githubusercontent.com/nvm-sh/nvm/v0.39.1/install.sh | bash
    #  Feche e abra o terminal novamente
    $  nvm install --lts # 16.14.0
    #  Verificar a versão do node
    $  node --version # Deve exibir v16.14
    #  Verificar a versão do nvm
    $  nvm -v
	
```
	
```bash
	
    # Clonar o repositório
    $ git clone https://github.com/WillianBL99/ZapRecall.git

    # Entrar no diretório
    $ cd ZapRecall

    # Instalar as dependências
    $ npm install

    # Iniciar o projeto
    $ npm start
```
